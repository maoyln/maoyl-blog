HTML+CSS 实现两只 圆锥渐变 合并形成的光台样式

  1. 镜面水平翻转法
<html>
  <div id="tBox1">
    <div class="ta" id="t1"></div>
    <div class="ta" id="t2"></div>
  </div>
</html>
<style>
    #tBox1 {
        display: flex;
    }
    
    .ta {
        width: 50%;
        padding-bottom: 56.25%;
        background: conic-gradient(from 90deg, blue, #6cf);
    }
    
    #t2 {
        transform: rotateY(180deg);
    }
</style>
